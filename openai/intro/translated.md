Introducing our developer blog

作者：Lucie Lozinski

我们为使用 OpenAI 构建的开发者推出了技术深度探讨、发布说明和最佳实践的新家。一个让我们的工程师直接与你谈论我们的工具和功能的地方。

推出博客

当我们发布新模型或 API 功能时，我们经常希望突出一些技术细节或提供额外上下文。不完全是文档，不完全是变更日志——把它想象成我们工程团队的笔记。我们将发布长篇文章，帮助在你集成时为你的工具和更新构建框架。

我们也有模型和 API 平台之外的开发者资源——仪表板功能、Codex 等。我们希望我们的写作帮助你发现这些工具并构建使用它们的强大心智模型。

除了这篇文章，我们的第一个帖子今天发布：开发者关于 Realtime API 的笔记。它为任何集成 GA Realtime API 和新实时模型的人强调了一些重要的技术更改。

谁适合阅读

这个博客面向 OpenAI 开发者。任何使用 OpenAI 平台开发的人——API、我们的模型或其他开发者工具——都被鼓励关注。

你想让我们写更多关于什么？你希望什么样的内容来帮助你在 OpenAI 上构建？我们很乐意听到你的想法。使用开发者社区论坛或 X 上的 @OpenAIDevs 提供反馈。

更多内容即将推出

今天，我们有前两篇文章：你正在阅读的这篇和我们关于 Realtime API 的开发者笔记。看看吧，看看你的想法，并继续关注未来的笔记和深度探讨。
